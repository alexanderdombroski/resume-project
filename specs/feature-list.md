# Resume Project

Resume Builder Website: CV -> Resume

_"Build once, tailor forever."_

**Required**

- OAuth to associated resumes with a person
- Resume editor
    - Choose recommended fonts
    - Double click fields to edit (rich text)
    - Resume meta (space between sections), format editor
- Resume printing - use print styles
    - Export to pdf through print
    - Indicator if resume is correct height
        - JS & Print stylesheet with units of inches
        - or use [PreText](https://www.npmjs.com/package/@chenglou/pretext) (browser only library)
- Resume viewer
    - Expanded resume CV editor: make all variations of resume in one document and fine tune after recording everything

**Extra (for future development)**

- Save resume templates
- Export to Markdown
- Basic Grammer Detection
- Autosave

**If ever made into a SaaS (for future development)**

- AI job posting analysis
- Export to DOCX
- Export to PDF
- Export to HTML
- Import a resume
- Import bullets from LinkedIn
- Create a handle where you can share your HTML resume
- STAR bullet point verification
- Team accounts (career coaches / recruiters), shared editing access
